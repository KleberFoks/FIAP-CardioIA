## 📜 Descrição
O projeto **CardioIA** é uma plataforma digital inteligente criada para simular o ecossistema de uma cardiologia moderna. Ele integra dados clínicos, modelos de Machine Learning, Visão Computacional, IoT e agentes inteligentes para lidar com triagem, diagnósticos, monitoramento e assistência remota. 

Nesta primeira fase, o foco é a **Busca de Dados**, estabelecendo uma base sólida de dados numéricos, textuais e visuais para alimentar os algoritmos nas fases subsequentes.

## 📁 Estrutura de pastas

- <b>assets</b>: arquivos relacionados a elementos não-estruturados, como imagens e logos.
- <b>config</b>: arquivos de configuração do projeto.
- <b>document</b>: documentos do projeto, incluindo os artigos médicos (.txt) para NLP e a documentação principal.
- <b>scripts</b>: scripts auxiliares.
- <b>src</b>: código fonte (desenvolvido ao longo das fases).
- <b>README.md</b>: guia e explicação geral sobre o projeto.
